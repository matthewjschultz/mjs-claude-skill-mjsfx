# Security Policy

## Supported versions

Only the current `main` branch is supported; there are no maintained older versions.

## Reporting a vulnerability

This repo is a Claude Code skill (instructions and reference material for designing sounds
with MJSFX's MCP tools) — it has no build step and runs no code of its own. If you find a
vulnerability here, or in the MJSFX app itself, please report it privately using GitHub's
**Report a vulnerability** button on this repo's
[Security tab](https://github.com/matthewjschultz/mjs-claude-skill-mjsfx/security/advisories/new),
or via [mjsfx.app/support](https://mjsfx.app/support).

Please do not open a public issue for security reports.
